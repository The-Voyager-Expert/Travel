DAY-COUNT SENTINEL VALUE MISMATCH + SPURIOUS SENTINEL
Added to validate_itinerary.py · warn-tier · 2026-07-21

What it catches: Two related drift patterns in <!-- day-count: N — reason --> sentinels:

(1) Value mismatch: Sentinel says N stops, but the actual stop count differs. Triggered when the crib uses the day number as N (e.g. Day 4 → day-count: 4) instead of the stop count (e.g. 2 stops → day-count: 2).

(2) Spurious sentinel: Day has ≥4 stops and doesn't need a sentinel at all — the sentinel from an earlier draft was not removed.

Test results: Brussels Day 4 (sentinel 4, actual 3) → fired. Brussels Day 2 (sentinel 3, actual 4) → fired. Brussels Day 3 (sentinel 3, actual 3, correct keyword) → no fire. Zhangjiajie Days 1/3/4 (sentinel 1/3/4, actual 2 each) → fired.
