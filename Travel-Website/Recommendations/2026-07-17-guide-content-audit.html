Vigadó Concert Hall — map URL contains typo across 3 locations

Google Maps query string used Vigan%C3%B3 in the encoded place name, which resolves to "Viganó" — a non-existent location. The correct encoding is Vigad%C3%B3 (Vigadó Concert Hall, Vigadó tér).

Appeared in three link targets: the stop's Google Maps search link (line 284), the address stop-row (line 463), and the Vigadó tour booking URL (line 455).

✓ All three occurrences corrected to Vigad%C3%B3. URLs re-verified post-fix.
