I built

  • new hard-fail GETTING AROUND — TRAM TRANSIT-BOX 3-ROW SHAPE (§2b)
    What it validates: every real 🚎 Tram .transit-box must hold exactly 3 child divs in order — (1) ↳ [Description] ≤160 chars with no <a> and a capital letter after the ↳, (2) the locked "[City] has a tram system — operated by [Operator]. Line(s) used on this trip: [N]." row, (3) the operator link row with an <a href>. Negative-finding entries ("No tram found in [City].") are excluded.
    Why it was needed: tram was the only Getting Around subsection with no row-count gate. Metro (Check D, §3b) and Ferry (§4b) have enforced an exact 3-child-div shape for months; tram had only a row-1 description check, a body-wide template search (TRAM TEMPLATE ADHERENCE matches the sentence anywhere in the box) and a link-presence check (any <a href> anywhere in the box). None of the three counts rows, so an arbitrary number of extra rows could sit between them and every existing tram check still passed. Row-1 "no link" and "capital after ↳" were added at the same time for exact parity with §3b/§4b, which already require both.
    Fleet impact: 1 guide — Edinburgh, which ships a 4th row (Board at … ride … alight at …). That is the .next-tram motion sub-line format, not a transit-box row; its "Board at" anchor also renders a Google Maps URL with the display text google.com. The other 14 tram guides were already 3-row clean.
  • new hard-fail GETTING AROUND — TRAM/METRO/FERRY OPERATOR LINK TEXT IS A DOMAIN
    What it validates: reads the last row of every 🚎 / 🚝 / 🚢 .transit-box — the operator link row per §2b/§3b/§4b — and hard-fails any <a> whose visible text contains an uppercase letter. The all-lowercase test is the same one §1b uses, chosen because the full domain set can't be enumerated.
    Why it was needed: the CORE RULES close all four subsection entries with the identical link-row construct — [Ride App Operator] (§1b), [Tram Operator] (§2b), [Metro Operator] (§3b), [Ferry Operator] (§4b). The 2026-07-12 pass established that this placeholder means the visible text is the operator's lowercase domain and enforced it for §1b only. The other three use the same construct and were never checked, so an operator's proper name or a domain-plus-description shipped in a tram/metro/ferry link row while the identical drift hard-failed one subsection above it.
    Fleet impact: 3 ferry guides — Boston (mbta.com — Inner Harbor Ferry.), Istanbul (Şehir Hatları), Stockholm (SL). 23 of the other 26 ferry guides, all 22 metro guides and all 15 tram guides already conformed — the drift is the exception, not the standard.
