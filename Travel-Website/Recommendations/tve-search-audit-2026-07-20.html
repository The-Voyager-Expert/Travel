Fix Applied — Best-Of Sub Truncation (34 pages)
FIXED — 34 pages All Best-Of pages — sub field contained full 3–4-part tag, causing false-positive matches on buried country/province names
Travel-Website/Trip-Essentials/Best-*.html (34 files)

Problem: Each Best-Of page passed the full .showcase-tag text as the sub field in its TVESearch items. Tags can be 3–4 comma-separated parts (e.g. "Qualicum Beach, Vancouver Island, British Columbia, Canada"). TVESearch matches on word-starts across both name and sub, so typing "Canada" or "British" would surface results where that word appeared anywhere in the full tag chain — not a useful match on buried province/country text.

Fix (Pattern A — showcase-card block, all 34 pages):

- if(n&&n.textContent.trim()) items.push({name:n.textContent.trim(), sub:t?t.textContent.trim():''});
+ var fullTag=t?t.textContent.trim():''; + var shortSub=fullTag.split(',').slice(0,2).join(',').trim(); + if(n&&n.textContent.trim()) items.push({name:n.textContent.trim(), sub:shortSub});

Fix (Pattern B — best-of-card fallback, Best-Of-Index only):

- if(n&&n.textContent.trim()) items.push({name:n.textContent.trim(), sub:t?t.textContent.trim():'', href:a?a.getAttribute('href'):''});
+ var fullTag=t?t.textContent.trim():''; + var shortSub=fullTag.split(',').slice(0,2).join(',').trim(); + if(n&&n.textContent.trim()) items.push({name:n.textContent.trim(), sub:shortSub, href:a?a.getAttribute('href'):''});
