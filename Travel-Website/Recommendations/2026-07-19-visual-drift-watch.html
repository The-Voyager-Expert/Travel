How to add the first reference screenshot
  1. Open a shipped guide in Chrome. Navigate to the section you want to capture (e.g., scroll to the "Also on this site" block).
  2. Take a screenshot of the section. Name it descriptively: YYYY-MM-DD-feature-name.png (e.g., 2026-07-19-also-on-site-pills-mobile.png).
  3. Drop the PNG in Brain/Reference/Visual-Reference/intake/.
  4. Create a companion .txt file with the exact same base name. Describe: what section it shows, what the correct HTML/CSS patterns are (one per line), and which tool currently enforces it.
  5. On the next visual-drift-watch run, the routine ingests it, moves it to canonical/, and adds it to the manifest. From that run onward, every newly shipped guide is compared against it.
