§ 3. Extras row

After the last day card, a compact pill row points to every extra section that ships in the guide. Section list and order: Guide Structure.html.

Each pill carries a canonical label — owner-defined, not a formatting choice. The validator hard-fails on any deviation. Pills that do not ship for a given guide are omitted; the label for every pill that does ship must be exact:

Pill labels are short navigation labels — they are not the section header titles. The full canonical section header (used in the guide HTML .extras-title) is defined in each section's own rule file. Example: the pill is 🎭 Shows but the section header is 🎭 Shows, Performances & Concerts; the pill is ⛲️ Day Trips but the section header is ⛲️ Day Trips by Train. § 4. Navigation anchors only

Day cards and extras pills are navigation anchors only. No global text-transform on anchors — it would lowercase proper nouns and tour names. The validator enforces this on the companion CSS file.
